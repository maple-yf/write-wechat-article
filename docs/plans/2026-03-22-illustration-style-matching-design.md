## 一、功能概述

为微信公众号文章写作技能增加插图风格智能匹配功能，根据文章内容自动推荐合适的视觉风格和色彩方案。

### 核心特性

- **AI 推荐 + 用户确认**：AI 分析后推荐 1-3 种风格，用户确认后应用
- **全文分析 + 关键词**：综合分析全文内容和关键词频率进行匹配
- **完整分析报告**：返回风格名称、色值、推荐理由、触发关键词
- **混合组合方式**：插图类型决定结构，风格提供视觉样式，可自由组合

## 三、匹配算法

### 3.1 算法流程

```
输入：文章标题 + 正文内容

Step 1: 关键词提取
- 合并标题和正文
- 分词（按空格、标点、中文分词）
- 统计每个关键词的出现频率

Step 2: 计算风格得分
- 对每种风格，计算得分 = Σ(关键词频率 × 权重)
- 例如：文章出现"代码"5次，暗色科技风得分 += 5 × 3 = 15

Step 3: 排序和筛选
- 按得分降序排列
- 过滤掉得分为 0 的风格
- 取前 3 种风格作为推荐

Step 4: 生成分析报告
- 对于每种推荐风格，列出触发关键词及次数
- 生成推荐理由

输出：推荐风格列表（1-3种）+ 分析报告
```

## 六、技术实现要点

### 6.1 关键词权重说明

- **权重 3**：强相关关键词，匹配后大幅提升得分
- **权重 2**：中等相关关键词
- **权重 1**：弱相关关键词，作为辅助判断

### 6.2 色值应用方式

1. 读取对应的模板文件
2. 替换模板中的 CSS 变量为选定风格的色值
3. 注入字体和特效样式
4. 生成最终插图 HTML

### 6.3 用户交互流程

1. AI 展示分析报告
2. 用户选择风格（输入编号或名称）
3. AI 应用选定风格的色值生成插图
4. 如果用户不满意，可以重新选择风格
